Box Shadowing

  • box-shadow - adds one or more shadows to an element our box-shadow is 10px 10px 5px and grey
    1. 10px - how far to push the shadow horizontally from the element
    2. 10px - how far to push the shadow vertically from the element
    3. 5px - blur-radius and spread-radius (optional)
    4. grey - color

Opacity

Another useful CSS property is opacity. A value of 1 is opaque, which isn't transparent at all. A value of 0.5 is half see-through. A value of 0 is completely transparent.

ex: opacity: 0.5;

relative

offsets of top or bottom, and left or right tell the browser how far to offset an item relative to where it would sit in the normal flow of the document

relative postion top:10px; right:20px;

Absolute

absolute locks the element in place relative to its parent container. surrounding items ignore it.

absolute postion top: 1750px; right:50px;
Element with a fixed position won't move when the user scrolls
