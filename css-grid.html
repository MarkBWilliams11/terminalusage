Other measurements to use

You can use absolute and relative units like px and em in CSS Grid to define the size of rows and columns.

  • fr: sets the column or row to a fraction of the available space
  • auto: sets the column or row to the width or height of its content automatically
  • %: adjusts the column or row to the percent width of its container

Using the repeat function

Using the repeat function allows us to repeat a number of columns or rows specifiedand its value.

  • example: grid-template-row: repeat(50, 6fr);

Using the minmax function

used to set values to change the container size.

  • grid-template-columns: repeat(3, minmax(50px, 600px));
  1. repeat 3 times
  2. min width 50px
  3. max width 600px
